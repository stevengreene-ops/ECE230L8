## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer allows for separate inputs to be routed via some switch(es) to a single output. These could be incoming signals (of any width) to an output signal. 
### In plain English describe the function and use of a demultiplexer.
A demultiplexer does the opposite, where it takes in a single input, and allows to reroute it to multiple outputs based on a switch or multiple switches in our case.
